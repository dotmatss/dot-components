# Working Memory

Use this file for stable notes that should carry between tasks.

## Current Project Notes

- The app is a monochrome Angular UI reference.
- The repo favors reusable primitives over one-off page logic.
- Routed pages should demonstrate patterns that can be reused in future projects.
- Browser verification matters for UI and stateful control changes.

## Verification

- Run `cmd /c npm run build` after UI or routing changes.
- If a change affects forms or interactive state, verify the rendered page in the browser when possible.

## Things To Preserve

- The black-and-white visual system.
- Small, composable components.
- Explicit route pages for layouts and flows.
- Modern Angular standalone patterns.
